The function $F$ is defined by

$$F(z)=\int_{0}^{\infty} \frac{t^{z-1}}{(t+1)^{2}} d t$$

For which values of $z$ does the integral converge?

Show that, for these values,

$$F(z)=\frac{\pi(1-z)}{\sin (\pi z)} .$$
