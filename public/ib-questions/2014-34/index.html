#Groups, Rings and Modules
2014

Let RRR be a ring, MMM an RRR-module and S={m1,…,mk}S=\left\{m_{1}, \ldots, m_{k}\right\}S={m1​,…,mk​} a subset of MMM. Define what it means to say SSS spans MMM. Define what it means to say SSS is an independent set.

We say SSS is a basis for MMM if SSS spans MMM and SSS is an independent set. Prove that the following two statements are equivalent.

  1. SSS is a basis for MMM.

  2. Every element of MMM is uniquely expressible in the form r1m1+⋯+rkmkr_{1} m_{1}+\cdots+r_{k} m_{k}r1​m1​+⋯+rk​mk​ for some r1,…,rk∈Rr_{1}, \ldots, r_{k} \in Rr1​,…,rk​∈R.

We say SSS generates MMM freely if SSS spans MMM and any map Φ:S→N\Phi: S \rightarrow NΦ:S→N, where NNN is an RRR-module, can be extended to an RRR-module homomorphism Θ:M→N\Theta: M \rightarrow NΘ:M→N. Prove that SSS generates MMM freely if and only if SSS is a basis for MMM.

Let MMM be an RRR-module. Are the following statements true or false? Give reasons.

(i) If SSS spans MMM then SSS necessarily contains an independent spanning set for MMM.

(ii) If SSS is an independent subset of MMM then SSS can always be extended to a basis for MMM.
