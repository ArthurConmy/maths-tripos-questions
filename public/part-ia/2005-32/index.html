4.II.5E

Numbers and Sets
Part IA, 2005

What does it mean for a set to be countable? Show that Q×Q\mathbb{Q} \times \mathbb{Q}Q×Q is countable, and R\mathbb{R}R is not countable.

Let DDD be any set of non-trivial discs in a plane, any two discs being disjoint. Show that DDD is countable.

Give an example of a set CCC of non-trivial circles in a plane, any two circles being disjoint, which is not countable.
