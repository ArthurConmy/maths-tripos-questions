The variables Θ\ThetaΘ and Φ\PhiΦ are independent. The probability density function of Θ\ThetaΘ is constant on (0,2π)(0,2 \pi)(0,2π) and the probability density function of Φ\PhiΦ is of the form A(π−ϕ/2)A(\pi-\phi / 2)A(π−ϕ/2) where 0<ϕ<2π0<\phi<2 \pi0<ϕ<2π, and AAA is a constant.

(a) Find the value of AAA. Calculate the expected value and the variance of the sector angle Φ\PhiΦ. Explain briefly how you would simulate the random variable Φ\PhiΦ using a uniformly distributed random variable UUU.
