2.I.1H

#Number Theory
2005

Recall that, if ppp is an odd prime, a primitive root modulo ppp is a generator of the cyclic (multiplicative) group (Z/pZ)×(\mathbb{Z} / p \mathbb{Z})^{\times}(Z/pZ)×. Let ppp be an odd prime of the form 22n+12^{2^{n}}+122n+1; show that aaa is a primitive root  mod p\bmod pmodp if and only if aaa is not a quadratic residue mod ppp. Use this result to prove that 7 is a primitive root modulo every such prime.
