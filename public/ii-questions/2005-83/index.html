#Linear Analysis
2005

Let KKK be a compact Hausdorff space, and let C(K)C(K)C(K) denote the Banach space of continuous, complex-valued functions on KKK, with the supremum norm. Define what it means for a set S⊂C(K)S \subset C(K)S⊂C(K) to be totally bounded, uniformly bounded, and equicontinuous.

Show that SSS is totally bounded if and only if it is both uniformly bounded and equicontinuous.

Give, with justification, an example of a Banach space XXX and a subset S⊂XS \subset XS⊂X such that SSS is bounded but not totally bounded.
