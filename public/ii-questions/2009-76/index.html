#Graph Theory
2009

(a) State Brooks' theorem concerning the chromatic number χ(G)\chi(G)χ(G) of a graph GGG. Prove it in the case when GGG is 3-connected.

[If you wish to assume that GGG is regular, you should explain why this assumption is justified.]

(b) State Vizing's theorem concerning the edge-chromatic number χ′(G)\chi^{\prime}(G)χ′(G) of a graph GGG.

(c) Are the following statements true or false? Justify your answers.

(1) If GGG is a connected graph on more than two vertices then χ(G)⩽χ′(G)\chi(G) \leqslant \chi^{\prime}(G)χ(G)⩽χ′(G).

(2) For every ordering of the vertices of a graph GGG, if we colour GGG using the greedy algorithm (on this ordering) then the number of colours we use is at most 2χ(G)2 \chi(G)2χ(G).

(3) For every ordering of the edges of a graph GGG, if we edge-colour GGG using the greedy algorithm (on this ordering) then the number of colours we use is at most 2χ′(G)2 \chi^{\prime}(G)2χ′(G).
