#Automata and Formal Languages

(a) Define the halting set K\mathbb{K}K. Prove that K\mathbb{K}K is recursively enumerable, but not recursive.

(b) Given A,B⊆NA, B \subseteq \mathbb{N}A,B⊆N, define a many-one reduction of AAA to BBB. Show that if BBB is recursively enumerable and A⩽mBA \leqslant_{m} BA⩽m​B, then AAA is also recursively enumerable.

(c) Show that each of the functions f(n)=2nf(n)=2 nf(n)=2n and g(n)=2n+1g(n)=2 n+1g(n)=2n+1 are both partial recursive and total, by building them up as partial recursive functions.

(d) Let X,Y⊆NX, Y \subseteq \mathbb{N}X,Y⊆N. We define the set X⊕YX \oplus YX⊕Y via

X⊕Y:={2x∣x∈X}∪{2y+1∣y∈Y}X \oplus Y:=\{2 x \mid x \in X\} \cup\{2 y+1 \mid y \in Y\} X⊕Y:={2x∣x∈X}∪{2y+1∣y∈Y}

(i) Show that both X⩽mX⊕YX \leqslant_{m} X \oplus YX⩽m​X⊕Y and Y⩽mX⊕YY \leqslant_{m} X \oplus YY⩽m​X⊕Y.

(ii) Using the above, or otherwise, give an explicit example of a subset CCC of N\mathbb{N}N for which neither CCC nor N\C\mathbb{N} \backslash CN\C are recursively enumerable.

(iii) For every Z⊆NZ \subseteq \mathbb{N}Z⊆N, show that if X⩽mZX \leqslant_{m} ZX⩽m​Z and Y⩽mZY \leqslant_{m} ZY⩽m​Z then X⊕Y⩽mZX \oplus Y \leqslant_{m} ZX⊕Y⩽m​Z.
